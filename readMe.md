# Tower Defense – TPSIT

Questa repository contiene la progettazione e lo sviluppo di un gioco Tower Defense non violento, realizzato come progetto scolastico

All’interno della repo sono presenti:

- Documentazione della storia del gioco
- Definizione degli elementi essenziali del gameplay
- Progettazione dell’interfaccia e della struttura dell’applicazione
- Suddivisione del lavoro tra i membri del gruppo
- Materiale per la presentazione finale
- Codice sorgente del gioco

La repo raccoglie sia la parte teorica di progettazione sia l’implementazione pratica del gioco

# Repo Structure 

-> docs/ProgettazioneTowerDefence.pdf (file dispensa del professore)

-> docs/Storia.sdocx (file modificabile Samsung Notes, con la lore)

-> docs/Storia.pdf (versione pdf della lore, visualizzabile da tutti)
